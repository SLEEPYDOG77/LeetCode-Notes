# 题目

Given a string, find the first non-repeating character in it and return it's index.

If it doesn't exist, return -1.



# 示例

### Example 1:

```
s = "leetcode"
return 0.

s = "loveleetcode",
return 2.
```



# 说明

You may assume the string contain only lowercase letters.
